Atom 编辑器使用

发表于2019-07-16|更新于2019-10-15|工具其他
|字数总计:801|阅读时长:2分钟|阅读量:

Atom 是 Github 专门为程序员推出的一个跨平台文本编辑器。具有简洁和直观的图形用户界面，并有很多有趣的特点：支持CSS，HTML，JavaScript等网页编程语言。它支持宏，自动完成分屏功能，集成了文件管理器。

  • 开源 — 遵循 MIT 协议,代码托管在 Github 上。
  • 多平台 — 支持 MAC/WIN/LINUX(支持源码编译安装,也提供二进制安装包)。
  • 丰富的插件库 — 开源到现在一年了..社区的各种插件丰富起来了,且 Atom 的插件支持在线更新。
  • 类 Sublime — 风格和 sublime text 极其相似,不管是风格还是操作上,快捷键上一些是通用的,应该借鉴
  • 采用包管理技术 — 采用了 node.js 来访问文件系统和包管理。
  • 强大的生命力 — 背靠 Github 社区,这对于 Atom 来说,可以注入源源不断的生命力。

教程

ATOM常用插件推荐

  1. simplified-chinese-menu：ATOM的汉化插件
  2. file-icons：文件图标美化
  3. atom-beautify：代码格式一键美化
  4. last-cursor-position：光标自由切换到上一次/下一次编辑位置
  5. minimap：代码小地图
  6. sync-settings：插件备份、按键绑定备份
  7. activate-power-mode：输入时有震撼效果
  8. tortoise-svn：svn插件
  9. atom-terminal-panel：atom内置命令行工具
  10. Git/github
  11. git-plus—在 Atom 里面执行 Git 命令，不用来回切换终端和编辑器
  12. git-control—git面板
  13. tree-view-git-status—文件夹git状态
  14. gist-it—快速分享代码到gist.github.com
  15. git-log：图形化git提交记录
  16. markdown-preview-plus—markdown预览
  17. vim-mode：支持vim模式
  18. script：在atom下运行脚本，支持多种开发语言。
  19. remote-edit：编辑机远程文件
  20. browser-plus：编辑器内置浏览器
  21. ask-stack：在atom快速提问stackoverflow
  22. regex-railroad-diagram：正则表达式图形化
  23. advanced-open-file：快速的打开文件或新建文件，同时支持路径补全
  24. quick-highlight：代码高亮
  25. platformio-ide-terminal：终端工具
  26. open-in-browsers：在安装过的任意浏览器打开代码
  27. merge-conflicts：在 Atom 里面处理合并产生冲突的文件
  28. linter：代码校验工具
  29. autocomplete-paths：填写路径的时候有提示
  30. linter：代码校验工具(必备)
  31. 前端插件推荐
  32. emmet：加快web开发速度，提供snippet(代码片段)、abbreviation expand(简写展开)功能。
  33. color-picker：在编辑器里面挑选颜色
  34. language-JavaScript-jsx：支持JSX语法
  35. atom-html-preview：实时预览HTML页面
  36. react—语法支持
  37. atom-react-native-autocomplete—React Native代码补全
  38. nuclide—facebook基于atom开发的进行react开发的IDE
  39. language-babel支持JSX语法
  40. atom-minify：css、js代码压缩
  41. atom-ternjs：js,nodejs,es6补全，高度定制化
  42. jshint：验证js和jsx语法是否正确
  43. csslint：验证css语法是否正确
  44. autoprefixer：自动为 CSS 属性添加特定的前缀
  45. autoclose-html：闭合html标签
  46. pigments：颜色显示插件
  47. less-autocompile：less文件编译为css文件
  48. php-class-tree：php类文件下函数展示
  49. php-cs-fixer：php语法检查
  50. python插件推荐
  51. python-tools
  52. Python跳转插件
  53. python-debugger:python debug
  54. autocomplete-python:python代码自动补全
  55. linter-pep8:python编码规范检查
